#proyecto salsamentaria
#descripcion de proyecto
## Tienda en línea: Los usuarios pueden ver los productos, filtrarlos por categoría, consultar su stock y agregarlos al carrito de compras.

## Panel de administración (Dashboard): Muestra estadísticas de productos, categorías y elementos en el carrito.

## CRUD de productos: Permite crear, editar y eliminar productos, incluyendo detalles como nombre, descripción, precio, categoría y stock.

## CRUD de categorías: Permite administrar las categorías de productos, con control de integridad para no eliminar categorías asociadas a productos.

## Autenticación básica: Login para el administrador (admin/admin123) que habilita las opciones de administración.
